# ⏱ Time Tracker (macOS)

A simple and lightweight time tracking app for macOS to help you stay productive and understand how you spend your time.

---

## 🚀 Features

* 🕒 Track time for different tasks and activities
* 📊 View time logs and history
* ⚡ Lightweight and fast macOS app
* 🎯 Designed for simplicity and focus
* 💻 Native macOS experience

---


---

## 📦 Installation

### Option 1: Download (Recommended)

Download the latest `.dmg` file from the Releases section.

### Option 2: Build from source

```bash
git clone https://github.com/jaikrishn-j/Time-Tracker-Macos.git
cd Time-Tracker-Macos
```

Open the project in Xcode and run the app.

* Swift / SwiftUI
